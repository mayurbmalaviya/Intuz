## Installation

Install all npm

```bash
npm install
```

### Setup your credential in config development file
  
### Create Database and table:
 
-  CREATE DATABASE intuz;
-  CREATE TABLE User
(id INT(11) primary key auto_increment,
firstName varchar(25),
lastName varchar(25),
email varchar(25),
phone varchar(25),
password text,
birthdate text,
gender varchar(20),
type varchar(20),
address varchar(20),
hobby text)
